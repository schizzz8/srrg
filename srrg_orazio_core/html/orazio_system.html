Parameter Set Field Explanation
motor_mode Type of H bridge.
0: for PWM and DIR
1: Disjoint PWM Channels
watchdog_cycles Watchdog control. Stops the robot if no communication occurs.
0: disable watchdog, orazio remembers the last command forever.
N: detach motors after N cycles
comm_cycles How many cycles of communication to skip.
Higher values mean reduced packet rate. The rate is 100/n
1: 100 hz
2: 50 hz
3: 33 hz
and so on often to perform the communication loop.
